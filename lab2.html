Usability

Here are a couple of recomendations to improve the performance of this site.

  • Compress components with gzip, which will reduce response times.
  • Avoid url redirects as they will slow the load time of the HTML documents.

Accessibility

Here are a couple of recomendations to improve the accessibility of this site.

  • Language is missing or was not recognized. This is a problem because the language attribute is used to let screen readers read the appropiate language
  • Various sections of the page are missing alternative text. Without alternative text, the content of an image will not be available to screen reader users or when the image is unavailable.
